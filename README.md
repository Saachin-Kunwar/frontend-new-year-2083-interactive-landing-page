🎆 New Year 2083 Interactive Landing Page

An interactive, animation-rich New Year landing page built using HTML, CSS, and Vanilla JavaScript.
This project focuses on frontend animation systems, Canvas rendering, and UI motion design principles.

📌 Overview

This project was created as a frontend engineering exercise to explore:

1.Canvas-based particle systems (fireworks animation)
2.CSS-driven interactive UI effects
3.SVG animation techniques
4.Layered UI composition without frameworks
5.Smooth motion design and timing control

The goal was to build a visually engaging experience using only core web technologies.

🚀 Live Features
1.🎆 Fireworks Particle System
Built using HTML5 Canvas API
Dynamic particle generation with random physics
Color variation using HSL model
Frame-synced animation using requestAnimationFrame

2.🖱️ Mouse-Based Reveal Interaction
Real-time mouse tracking
CSS variables (--x, --y) for dynamic updates
Circular reveal effect using clip-path
Smooth transition-based motion

3.⌨️ Typing Animation Header
CSS keyframe-based typewriter effect
Cursor blinking animation
Delayed entry for staged UI reveal


4.✍️ SVG Signature Animation
Stroke drawing animation using:
stroke-dasharray
stroke-dashoffset
Delayed animation sequencing
Glow effect using CSS filters

5.🧩 Layered UI Architecture
Canvas background layer (fireworks)
Mid-layer content (context + banner)
Foreground UI (greeting + signature)
Controlled stacking using z-index


🛠️ Tech Stack
HTML5
CSS3 (Animations, Clip-path, Flexbox, SVG styling)
JavaScript (ES6+)
Canvas API
DOM Events
Animation loops

4.Structuring layered frontend architectures
